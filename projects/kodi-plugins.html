Kodíček

2025 · Role: návrh + kód (Kodi addon) · Stack: Python / Kodi API / TMDB / Webshare

Minimalistický Kodi plugin pro pohodlné vyhledávání a přehrávání filmů a seriálů z uživatelského účtu Webshare.cz. Soustředí se na chytré párování výsledků, historii přehrávání a čisté UX v češtině.

Co to řeší

Běžné doplňky často zobrazují dlouhé seznamy nejasně pojmenovaných souborů. Kodíček nejdřív najde titul přes TMDB a teprve potom k němu hledá odpovídající video soubory na tvém vlastním Webshare účtu. Plugin žádný obsah sám nešíří ani neukládá; pracuje jen s tím, k čemu máš přístup ty (přihlášení do Webshare).

Tok uživatele

  1. Spuštění doplňku → „Vyhledat film/seriál“.
  2. Zadáš název (např. „Spider-Man 3“, „Pán prstenů 2001“).
  3. TMDB vrátí titul, rok, popis, poster.
  4. Vybereš titul → nabídnou se zdroje z Webshare (u seriálu série/epizody).
  5. Klik → Kodi přehrává video ze tvého Webshare účtu.

Klíčové funkce

  • TMDB integrace: vyhledávání filmů i seriálů, metadata a obrázky.
  • Chytré párování Webshare: normalizace názvů, rok, kvalita, jazyk; filtrování jen na video soubory.
  • Historie: lokální JSON pro přehrávané tituly i vyhledávání.
  • Seriály: série → epizody, dotaz na Webshare podle S/E.
  • UX & CZ: kompletně česky, hlášky pro chybové stavy.

Technologie

  • Kodi plugin (Python: xbmc, xbmcplugin, xbmcgui).
  • Webshare API pro práci s uživatelským účtem.
  • TMDB API pro metadata a obrázky.
  • Lokální JSON (history.json, search_history.json).
  • Připraveno na integraci s Up Next.

Limity

  • Funguje pouze s platným přihlášením k tvému Webshare účtu.
  • Nezaručuje dostupnost konkrétního titulu; záleží na výsledcích Webshare.
  • Neobsahuje vlastní katalog obsahu, nic nešíří.

Simpsonovi (Elementum helper)

2025 · Role: návrh + kód (Kodi addon) · Stack: Python / Kodi API / Elementum / bencode

Malý specializovaný doplněk, který z jednoho velkého torrentu se seriálem udělá přehledné menu „Série → Epizody“. Nesahá na síť, jen čte lokální torrent metadata a předává správný index do doplňku Elementum, které řeší samotné přehrávání.

Co to řeší

U vícesouborového torrentu končíš v přehrávači s dlouhým bezejmenným seznamem. Tenhle doplněk nad Elementum staví jednoduché rozhraní: vybereš sérii/epizodu a on k ní dohledá správný soubor.

Tok uživatele

  1. Spuštění doplňku „Simpsonovi“.
  2. Seznam sérií 1–34 → epizody.
  3. Klik na epizodu: podle map.json se najde index souboru v torrentu a sestaví se plugin://plugin.video.elementum/… URL.

Klíčové funkce

  • Přehledné menu (série → epizody).
  • Mapování epizod přes map.json.
  • Integrace s Elementum: doplněk nic nestahuje, jen předává vstup Elementu.
  • Lokální provoz: čtení .torrent metadat přes bencode.

Technologie

  • Kodi plugin (Python: xbmc, xbmcplugin, xbmcgui, xbmcaddon).
  • bencode pro parsování .torrent.
  • Integrace s plugin.video.elementum přes plugin:// URL.

Limity

  • Navázané na konkrétní strukturu lokálního .torrent souboru (případně upravit map.json).
  • Bez nainstalovaného a nastaveného Elementum nepřehraje nic.
  • Žádný katalog/obsah; čistě technická nadstavba pro osobní archiv.

Stažení

ZIPy jsou k dispozici zde: /kodi/

Instalace ze ZIP

  1. Nastavení → Systém → Doplňky: povol Neznámé zdroje.
  2. Doplňky → Instalovat ze souboru ZIP → vyber stažený plugin.video.*.zip.
  3. Doplněk najdeš v Doplňky → Video doplňky.

Sekce /kodi/ je záměrně mimo navigaci a s noindex, nofollow. Použití doplňků je na vlastní zodpovědnost.
